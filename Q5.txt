Normaly we cannot use return instead of break if we use it will give an error 'Illegal return statement'.
Although if we are using loop in a function then return can be use in place of break if we want to return to invoker of the method i.e. if a certain condition or requirement is met and we do not want to further execute anything then we can use return.
for example we are writing a code to find the first factor of a number after 1 then one way of writing code can be :
function firstFactor(n)
{
    let ans;
    for(let i=2 ; i<=n ; i++)
     {
        if(n%i==0)
         {
            ans=i;
            break;
         } 
     }
     return ans;
}
console.log(firstFactor(9));

Instead of using break we can simply return the value when the condition is met.

function firstFactor(n)
{
    let ans;
    for(let i=2 ; i<=n ; i++)
     {
        if(n%i==0)
            return i;
     }
}
console.log(firstFactor(15));
